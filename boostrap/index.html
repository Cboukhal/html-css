Bootstrap

Introduction

Bootstrap est un framework CSS. Un framework est un ensemble de librairies regroupées dans un but précis et possédant des règles internes que doivent suivrent les utilisateurs.
En d'autres termes et pour le dire plus simplement, Bootstrap est un ensemble de fichiers de CSS et Javascript en fontionnant ensemble et qu'on va pouvoir utiliser pour créer des designs complexes de manière relativement simple.
L'intêret d'un framework c'est de nous permettre de réaliser plus de chose en codant moins.

Comment utiliser Bootstrap ?

Bootstrap est un ensemble de fichier CSS et Javascript, donc pour l'utiliser nous devons intégrer ces fichiers à notre projet ou tout simplement faire appel à ses fichiers via un CDN (Content delivry network).

  • Premiére méthode
    • Télécharger Bootstrap depuis le site https://getbootstrap.com/ ensuite intégrer les fichiers CSS et Javascript à notre projet comme n'importe quel autre fichier CSS ou Js.
  • Seconde méthode
    • Utiliser un CDN de bootstrap et le lier à notre projet comme n'importe quel autre CDN(réseau de distribution de contenu)
