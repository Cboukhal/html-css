2. Les balises texte

Je suis une balise paragraphe et je suis une balise bloc

je suis une balise span et je suis une balise inline je suis une balise span et je suis une balise inline Le span est une balise inline alors que le p est une balise bloc. Les balises bloc sont des balises egoistes, car elles ne partagent pas la ligne. Tant dis que les balises inline sont plutot altruiste, elles partagent la ligne.
La balise br permet le saut de ligne. Elle va etre generalement utilise avec les balises inlines.
La balise u permet de souligner
La balise i et em permet de mettre en italique
la balise strong ou bolder et meme b met en gras
la balise sub permet de mettre en indice
la balise sup permet de mettre en exposant
la balise s et strike permet de barrer le texte
NB: les balises obsoletes
la balise center permet de centrer un element

la balise big permet de grossir le texte
la balise small permet de reduire le texte

3. Les balises listes

3.1. les listes ordonnees

les balises listes fontionnent avec 2 paires de balises, une paire qui va definir si il s'agit d'une liste ordonne ou non, une autre qui va definir les element de la liste. Les balises liste sont des balises bloc.

  1. Premier
  2. Deuxieme
  3. Troisieme

On peut changer les tyupes de puces avec l'attribut type

liste avec type A:

  1. Premier
  2. Deuxieme
  3. Troisieme

liste avec type a:

  1. Premier
  2. Deuxieme
  3. Troisieme

liste avec type I:

  1. Premier
  2. Deuxieme
  3. Troisieme

liste avec type i:

4. L'ancre

La balise a comme ancre permet de creer des liens hypertexte pour faciliter la navigation entre les pages web. C'est une balise inline. Son attribut href doit contenir l'adresse du lien ou on souhaite envoyer l'utilisateur.
