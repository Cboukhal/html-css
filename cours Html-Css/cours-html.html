6. Attribut

Les attributs sont des elements qu'on rajoute à la balise html pour apporter des specifiquation supplementaires a l'elements cible.

  • LANG:permet de definir le langage avec lequel on programme. Exemple fr, en, ...
  • CHARSET:permet de definir l'encodage utiliser
  • NAME:permet de specifier en donnant un nom a la balise meta
  • CONTENT:permet de mettre du contenu a la balise meta
  • ID:permet de donner un identifiant unique a un element
  • CLASS:permet de regrouper certains specification a plusieurs elements
  • TARGET:permet de renvoyer un lien vers une nouvelle page
  • TYPE:permet de definir le type d'un element
  • ALT:permet de decrire une image
