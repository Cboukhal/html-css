10.Balises Css

10.1 Introduction

Le CSS (cascading style sheet) est un langage informatique qui permet de styliser les documents HTML. Il permet de controler l'apparence ou la presentation des pages HTML. Les standards definissants le CSS sont publiés par le W3C.

10.2 Objectifs

  • Separer le fond de la forme, c'est a dire separer la structure html des styles css.
  • Controler l'apparence et le rendu des pages html.
  • Controler exclusivement la forme et non le fond.
  • Interpretation et compatibilite avec les navigateurs.

10.3 Historique

  1. La premiere version du CSS a ete publiés en 1996.
  2. En 1997 internet explorer utiliser le Css.
  3. En 1998 le W3C publie la premiere version de CSS qui va devenir la norme de style conforme.

10.5 Principes

La synthaxe du css se compose d'un ensemble de selecteur suivi par un bloc d'instructions delimite par des accolades. Chaque instruction est compose d'une propriete suivi de : et d'une valeur et se termine par un point virgule (;)
Exemple:
selecteur(s) {propriete : valeur ; ...}

Les types de selecteurs

  • Elements
    • selectionneer a partir du nom ou des noms de balises
  • class
    • selectionneer a partir du nom de la classe precede d'un point(.),au moins deux elements ont cette meme classe
  • ID
    • selectionneer a partir du nom l'ID precede d'un #, identifiant ubnique pour un element

Hierarchie des selecteurs

  1. ID
  2. Class
  3. Element balise(s)

10.9 Media queries

Les medias queries font parties des nouveautes de CSS3. Il ne s'agit pas de nouvelles proprietes mais de regles que l'ont peut appliques dans certaine conditions. Concretement nous allons pouvoir dire si la resolution de l'ecran de l'utilisateur est inferieur a tant, applique ces proprietes CSS. Cela nous permet de changer l'apparence du site en fonction des tailles des ecran. C'est se qu'on appelle le responsive de design.

Synthaxe

Une requete media se compose d'un type de media optionnelle et d'une ou plusieurs expressions de caracteristiques de media. Plusieurs peuvent etre combines entre elles grace a des operateurs logiques. Les requetes media ne sont sensibles a la casse.
